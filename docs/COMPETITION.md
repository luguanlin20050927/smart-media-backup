# 影序 YINGXU｜项目展示摘要

## 一句话

影序是一款面向摄影师与视频创作者的本地素材归档工具：把“插卡、识别、分类、备份、校验、找回”连成一条可信赖的工作流。

## 要解决的问题

拍摄素材往往在卡里积累数天甚至数周。文件名、拍摄设备和项目上下文很快变得难以回忆；更危险的是，用户以为已经备份，实际却没有完成或无法确认。

## 核心方案

1. 从存储卡扫描媒体文件。
2. 从 EXIF、目录和文件名提取设备线索。
3. 按设备、事件、照片/视频建立可读的归档结构。
4. 使用增量策略跳过已存在文件，减少重复写入。
5. 对复制结果进行 SHA-256 校验，并保存历史记录与报告。
6. 失败时保留可解释状态，支持重试和历史检索。

## 可验证成果

- 真实 SD 卡：87 个文件，约 8.66 GB，复制与校验通过。
- 重复执行：87 个文件全部正确跳过。
- 设备归类：Leica 60 个、Fujifilm 27 个。
- 自动化测试覆盖 12 个核心场景，包括无卡、目标不可写、空间不足、损坏文件、网络不可用和重复备份。
- macOS Apple Silicon v1.0.10 DMG 可下载、安装并启动本地 Web 工作台。

## 展示建议

比赛演示用“真实素材卡 → 选择归档目标 → 开始备份 → 历史记录 → 第二次执行跳过 → 打开报告”六步完成。全程不需要云服务，也不会上传原始素材。

## 当前边界

当前最完整的是 macOS 体验版。Windows 安装包尚未在 Windows 真机验收；iOS、Android 原生应用和百度网盘自动上传仍是后续方向。

# 影序 YINGXU｜验收矩阵

| 场景 | 预期行为 | 状态 |
|---|---|---|
| 没有 SD 卡 | 页面说明未开始复制，原始素材安全 | 已验证 |
| 目标磁盘空间不足 | 前置检查阻止任务，并说明原因 | 已验证 |
| 目标目录不可写 | 前置检查阻止任务，并说明原因 | 已验证 |
| 重复备份 | 已存在文件跳过，不重复写入 | 已验证 |
| 损坏文件 | 失败计数、报告和历史状态一致 | 已验证 |
| 网络不可用 | 本地备份不依赖网络；可选通知失败不影响本地结果 | 已验证 |
| 进度重置 | 新任务不显示上一轮瞬时进度 | 已验证 |
| 部分失败 | 保留已完成结果，支持重试 | 已验证 |
| 真实 SD 卡 | 扫描、复制、SHA-256 校验、重复执行 | 已验证 |
| Windows 安装 | 需要 Windows 真机 | 未验证 |
| iOS / Android 原生端 | 需要移动端开发与设备 | 未完成 |

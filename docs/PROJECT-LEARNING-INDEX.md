# 影序 YINGXU 项目学习资料总索引

这组文档用于复盘从最初接手项目到 macOS v1.0.11 发布的全过程，目标是让下一次项目更快、更清楚、更少返工。

## 建议阅读顺序

1. [完整项目流程图](PROJECT-FLOWCHART.md)：先理解项目应该怎样推进。
2. [项目时间线与返工分析](PROJECT-TIMELINE.md)：再看这次实际发生了什么。
3. [Bug 与修复地图](BUG-AND-REWORK-MAP.md)：理解哪些错误出现在什么阶段，以及如何提前发现。
4. [个人项目经理学习清单](PERSONAL-PM-LEARNING-CHECKLIST.md)：把复盘转换成下一次的行动能力。

## 当前项目成果

- macOS v1.0.11 已发布。
- 官网已发布 v33 页面版本。
- GitHub Release、官网、DMG 和 SHA-256 已完成同步。
- 核心备份、校验、增量跳过、异常保护、历史记录和报告功能已经形成闭环。
- Windows、移动端正式版本和部分物理现场测试仍属于后续阶段。

## 复盘方法

文档中的结论分为三种：

- **事实**：有 Git 提交、测试结果、文件或线上页面作为依据。
- **判断**：根据返工路径得出的项目管理分析。
- **建议**：下一次项目应当采用的规则。

为什么需要自动备份？Why Auto Backup?

手动拖文件有几个致命问题：第一，拖到一半忘了校验，拷坏了都不知道；第二，不同设备（Sony + DJI）的文件混在一起，后期整理要花额外时间；第三，万一硬盘满了、或者不小心格式化了 SD 卡，数据就丢了。自动备份能解决所有这些问题：插卡、打字、点一下，剩下全自动。Manual file dragging has several fatal flaws. First, you might forget to verify copies — corrupted files go unnoticed. Second, files from different devices (Sony + DJI) get mixed together, requiring extra cleanup later. Third, if your drive fills up or you accidentally format the SD card, your data is gone forever. Auto backup solves all of this: insert the card, type an event name, click start — everything else is automatic.

Smart Media Backup 怎么帮你？How Smart Media Backup Helps

这个工具的核心逻辑很简单：The core logic is simple:

  • 检测—SD 卡插入读卡器，程序自动识别
  • Detection—SD card inserted, the app auto-detects it
  • 读取—自动提取相机型号、拍摄日期、GPS 坐标
  • Reading—Extracts camera model, shooting date, GPS coordinates
  • 分类—按设备→事件→照片/视频 自动建文件夹
  • Organizing—Auto-creates folders by device → event → photos/videos
  • 校验—SHA256 检查每份拷贝，保证零错误
  • Verification—SHA256 checks every copy for zero errors
  • 同步—可选百度网盘自动上传，本地云端双保险
  • Sync—Optional auto-upload to cloud, local + remote redundancy

使用步骤Usage Steps

  • 下载安装（macOS .dmg 或 Windows .exe）
  • Download and install (macOS .dmg or Windows .exe)
  • 双击启动，浏览器自动打开 Web 面板
  • Launch the app, the Web panel opens automatically
  • 插 SD 卡 → 输事件名 → 选目标盘 → 点开始
  • Insert SD card → enter event name → pick target drive → click start
  • 看进度条，备份完拔卡走人
  • Watch the progress bar, then eject and go

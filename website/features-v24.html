核心功能

插卡、打字、点一下，剩下全自动。从检测到校验，每个环节都是透明的。

🔌

插卡即检测

SD 卡插入读卡器，后台每 2 秒自动轮询，弹窗通知。支持 RAW、JPG、MP4、HEIC 等主流格式。

🏷️

EXIF 元数据提取

自动读取相机型号、拍摄日期、GPS 坐标。支持 Sony、Canon、DJI、GoPro、Nikon 等 100+ 设备。

📂

智能分类整理

按设备→事件→照片/视频三层自动分类。目录结构清晰，原始文件名保留不变。

🔒

SHA256 校验

每份文件拷贝后自动计算校验和，与源文件比对。零错误保障，拷贝失败自动重试。

🤖

AI 自动命名

插入 SD 卡后自动识别照片内容，建议事件文件夹名。支持本地 Ollama 或云端 OpenAI。

☁️

百度网盘同步

本地备份完成后自动触发百度网盘上传。OAuth 授权，分片上传，大文件不断连。

📊

实时可视化

Web 面板实时展示进度条、拷贝速度、剩余时间。多设备排队备份，互不影响。

📋

完整历史记录

每次备份写入 SQLite 数据库，可查询每份文件的来源、目标、校验状态和时间。

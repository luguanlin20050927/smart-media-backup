三步完成备份

你只需要做 3 件事，其余全自动

1

📥 插卡

SD 卡插入读卡器，程序自动检测并打开仪表盘。支持 Sony、DJI、GoPro、iPhone、Nikon、Canon 等主流设备。

2

📝 命名

输入事件名，比如"2025年8月漫展"。AI 会自动分析照片内容给出建议名。多个活动用逗号分隔即可分批备份。

3

▶ 开始

点一下按钮，自动分类整理到目标磁盘（支持同时备份到多个盘）。完事后自动校验、筛选废片、上传百度网盘、推送通知。

功能特性

摄影师需要的，这里都有

⚡ 核心流程

📷

自动读取元数据

相机型号、拍摄日期、GPS 坐标自动提取，不需要你手动填。支持 RAW(JPG/ARW/CR3/DNG等) + 视频(MP4/MOV/MXF等)。

📁

智能分类整理

按设备→事件→照片/视频三层目录，文件井井有条。一张卡多个活动？逗号分隔事件名即可分批备份。

🔒

SHA256 校验

每个文件拷贝后进行完整性校验，保证备份零错误。每目录自动生成 checksums.json。

📊

实时可视化仪表盘

进度条、拷贝速度、剩余时间一目了然。Web 面板即开即用，SocketIO 实时推送进度。

⏳

失败自动重试

拷贝失败、校验失败自动重试 3 次，确保每个文件安全到达。

🔄

增量备份

基于 SHA256 哈希去重，已备份过的文件自动跳过。第二次插同样的卡只需几秒。

📦 多目标 & 同步

💾

多目标备份

同时备份到多个位置：外置硬盘 + NAS + 本地磁盘。一次扫描，多处写入。

☁️

百度网盘自动上传

备份完成后后台自动上传到百度网盘。支持断点续传、大文件分片。配置一次永久有效。

🔄

跨电脑同步

多台电脑运行 SMB，备份历史自动同步。家里和工作室各一台，进度共享。

📤

通知推送

备份完成后通过 Webhook 推送通知到微信/Telegram/任意服务，不守着也知道。

🤖 AI 智能

🧹

废片筛选

自动识别过曝、过暗、模糊、黑图、白图、闭眼、构图差。废片不删除，移到「废片待确认」文件夹你自己决定。

🏷️

AI 自动命名

扫描 SD 卡后自动分析照片内容，给出 2-6 字中文事件名建议。支持 Ollama 本地模型或 OpenAI 云端。

🎯

感知哈希去重

连拍场景下，内容几乎相同的照片自动分组标记。基于 imagehash 感知哈希算法，可调敏感度。

🖥️ 平台 & 体验

🍎

macOS + Windows

macOS 原生 App（.dmg 28MB），Windows .exe 由 GitHub Actions 自动构建。均支持 Intel 和 ARM。

🪟

Windows 缩略图

备份完成的照片在 Windows 资源管理器里直接显示缩略图，不用逐个打开查看。

🌙

亮/暗主题

侧边栏一键切换亮色/暗色主题，偏好自动保存到本地存储。

📋

备份历史 + 搜索

每次备份记录在案，按时间/设备/事件名/状态筛选，详细文件列表和校验结果一键可查。

🐢

速度限制

设置最大拷贝速度，备份时不影响同时运行的 Lightroom 或 PS。不限速时全速拷贝。

📸

Lightroom 集成

备份完成后自动生成 Lightroom 导入指南和元数据 JSON，方便导入到 LR 目录。

🔧 安全 & 控制

🗑️

备份后清理 SD 卡

备份完成后询问是否删除 SD 卡文件。选「删除」则移到废纸篓，不彻底删除，给一次后悔机会。

📋

事件模板

旅行、婚礼、日常、商拍、活动等预设事件模板，一键填名不用打字。

📝

备份报告

每次备份生成详细 JSON 报告：成功/跳过/失败文件数、各设备统计、耗时、废片筛选结果。

🖼️

插卡即弹窗

SD 卡插入后自动打开浏览器到仪表盘，2 秒间隔扫描 /Volumes 检测新设备。

🪜 安装教程

无签名，按下面操作即可打开

🍎

macOS 安装

  1. 下载 SmartMediaBackup-macOS.dmg
  2. 双击 .dmg → 把 App 拖到 Applications
  3. 打开启动台 → 点 Smart Media Backup
⚠️ 首次打开弹「无法验证开发者」
这是因为本软件是开源项目，没有花钱买 Apple 开发者证书 ($99/年)。

解决办法：
  1. 去 系统设置 → 隐私与安全性
  2. 往下翻，看到 「Smart Media Backup 被阻止使用」
  3. 点 「仍然打开」
  4. 输入密码 → 以后直接双击就能用了
💡 备用方法：右键 App → 点「打开」→ 弹窗点「打开」，效果一样
🪟

Windows 安装

  1. 下载 SmartMediaBackup-Windows.exe
  2. 双击运行安装程序
  3. 安装完成后从桌面快捷方式启动
⚠️ 首次运行弹「Windows 已保护你的电脑」
因为本软件没有购买代码签名证书，Windows Defender 会拦截。

解决办法：
  1. 点击 「更多信息」（蓝色链接）
  2. 底部会出现 「仍要运行」 按钮
  3. 点 「仍要运行」
  4. 以后就不会再弹了
💡 安全说明：代码完全开源 (MIT)，可在 GitHub 审查源代码。
弹窗不是因为软件有问题，只是因为没买签名证书。

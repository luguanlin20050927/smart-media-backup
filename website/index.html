功能介绍

插卡、打字、点一下，剩下全自动

01

插卡即弹窗

SD 卡插入读卡器，后台每 2 秒轮询 /Volumes。检测到新设备自动打开浏览器到仪表盘。支持 RAW、JPG、MP4、MOV、HEIC 全格式识别。

📥 → 🌐
02

AI 智能分类

每张照片的相机型号、拍摄日期、GPS 坐标自动提取。按设备/事件/照片-视频三层分类。目录层级可自由排序：设备、日期、地点随你设。

📷 → 📁
03

多目标 + 云同步

同时备份到外置硬盘、NAS、本地磁盘。百度网盘后台自动上传（分片+断点续传）。多台电脑可同步备份历史。完成后 Webhook 推微信通知。

💾 → ☁️
04

AI 废片筛选

自动识别过曝、过暗、模糊、闭眼、构图差。废片移到待确认文件夹。连拍场景感知哈希自动分组相似照片，敏感度可调。

安装教程

无签名，也能用

🍎 macOS

  1. 下载 .dmg → 双击 → 拖到 Applications
  2. 打开启动台 → 点 App
⚠️ 首次弹「无法验证开发者」
系统设置 → 隐私与安全性 → 点「仍然打开」
💡 右键 App → 点「打开」，效果一样

🪟 Windows

  1. 下载 .exe → 双击运行
  2. 从桌面快捷方式启动
⚠️ 弹「Windows 已保护你的电脑」
点「更多信息」→「仍要运行」

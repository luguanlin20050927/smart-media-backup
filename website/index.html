三步完成备份

你只需要做 3 件事，其余全自动

1

📥 插卡

SD 卡插入读卡器，程序自动检测。支持 Sony、DJI、GoPro、iPhone 等主流设备。

2

📝 命名

输入事件名，比如"2025年8月漫展"。日期和设备型号程序自动读取。

3

▶ 开始

点一下按钮，自动分类整理到目标磁盘。完事后会通知你。

功能特性

摄影师需要的，这里都有

📷

自动读取元数据

相机型号、拍摄日期、GPS 坐标自动提取，不需要你手动填。

📁

智能分类整理

按设备→事件→照片/视频三层目录，文件井井有条。

🔒

SHA256 校验

每个文件拷贝后进行完整性校验，保证备份零错误。

📊

实时可视化

进度条、拷贝速度、剩余时间一目了然。Web 面板即开即用。

📋

备份历史

每次备份记录在案，随时可查详细的文件列表和校验结果。

💾

任意目标位置

本地磁盘、外置硬盘、NAS 都可以作为备份目标。

🪜 安装教程

一分钟上手

1

⬇ 下载安装包

点击上方「下载」按钮，下载 SmartMediaBackup-macOS.dmg（28MB）。

2

📂 安装到电脑

双击 .dmg 文件 → 把 Smart Media Backup 拖到 Applications 文件夹。

3

🚀 首次打开

打开启动台 → 点 Smart Media Backup。
⚠️ 第一次会弹「无法验证开发者」
去 系统设置 → 隐私与安全性 → 点「仍然打开」
以后直接双击就能用了。

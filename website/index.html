识别中设备与素材

识别来源、拍摄线索与文件类型。

BUILT FOR TRUST

素材安全
要能被证明

影序不把“放心”写成口号，而是把每一步做成可以检查的记录。
01 / SOURCE SAFE

原卡不动

影序读取并复制素材，不重命名、不修改、不删除 SD 卡中的原始文件。

02 / SHA256

复制后校验

通过 SHA256 对照源文件与目标文件，明确告诉你这次备份是否完整。

03 / HISTORY

留下一条记录

设备、项目、目标位置和完成状态写入历史，下一次找回不再从“我记得”开始。

INSTALL

安装
然后开始归档

影序优先服务本地工作流；你的素材、路径和历史都由你自己掌握。

macOS

  1. 下载 .dmg 并拖入 Applications
  2. 从启动台打开影序
首次如出现系统提示：前往「系统设置 → 隐私与安全性」选择仍要打开
也可以按住 Control 点击 App 后选择「打开」

完整性校验

  1. 下载后，在终端运行 shasum -a 256 文件名.dmg
  2. 确认结果与下载页公布的 SHA-256 一致

安装步骤

安装包在下载页提供 SHA-256；下载后先校验，再安装。
MACOS

macOS 安装

  1. 下载 v1.0.1 的 .dmg，打开后将「影序 YINGXU」拖入「应用程序」。
  2. 从 Applications 或启动台启动影序。
  3. 如被系统拦截，前往「系统设置 → 隐私与安全性」选择「仍要打开」。
这是 macOS 最常见的首次运行提示。
WINDOWS

Windows 发布状态

  1. 核心功能与打包配置已准备完成。
  2. 仍需在 Windows 环境执行构建与实际启动验证。
  3. 验证完成前，不提供 .exe 下载。
这是为了避免用户下载到未经验证的安装包。

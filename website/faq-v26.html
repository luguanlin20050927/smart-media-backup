常见问题

支持哪些相机？▼
支持 Sony、Canon、Nikon、DJI、GoPro、Fujifilm、Panasonic 等市场主流品牌。只要照片/视频包含 EXIF 元数据即可自动识别设备型号。未识别的设备会归类为「Unknown」。
备份到 NAS 怎么配置？▼
在软件设置页的「备份目标」下拉框中，选择已挂载的 NAS 目录即可。SMB/NFS 协议均可，只要操作系统能识别为本地目录。Web 面板会自动列出所有可用存储卷。
百度网盘怎么授权？▼
打开设置页 → 百度网盘 → 填 App Key 和 Secret Key（需在百度开放平台注册）→ 点「授权」→ 跳转到百度登录 → 同意授权 → 把授权码粘贴回来即可。之后备份完会自动上传。
会和 Lightroom / DaVinci 冲突吗？▼
不会。本工具只读取 SD 卡上的文件并复制到目标目录，不修改源文件、不占用文件锁、不干扰其他软件的正常使用。备份完成后会释放 SD 卡。
Windows / macOS 都能用吗？▼
都支持。macOS 提供 .dmg 安装包，Windows 提供 .exe 安装包（GitHub Actions 自动构建）。Linux 可通过 pip 安装。树莓派 Pi Zero 2W 也支持。
软件会读取我的隐私数据吗？▼
不会。代码完全开源（MIT 协议），所有数据处理都在本地完成。AI 命名功能如果使用本地 Ollama 模型则完全离线。百度网盘上传需要授权，但只会上传你指定的备份目录内的文件。

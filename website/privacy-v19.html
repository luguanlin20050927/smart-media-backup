隐私政策

Smart Media Backup（以下简称「本软件」）是一个开源的个人媒体备份工具。我们重视你的隐私，以下是本软件的数据处理说明。

1. 数据存储位置

所有数据处理均在本地完成。本软件不会将你的照片、视频或其他文件上传到我们控制的任何服务器。

2. 百度网盘上传

百度网盘上传功能需要你在设置页中手动授权。只有在授权并开启该功能后，本软件才会在本地备份完成后将文件上传到你个人百度网盘的指定目录。上传行为完全在你的控制之下，你可以随时关闭该功能。

3. AI 命名功能

AI 命名功能默认使用本地 Ollama 模型（llava），所有图像数据在本地处理，不会离开你的电脑。如果你选择使用 OpenAI 兼容 API，图像数据将被发送到你指定的 API 服务端处理。你可以在设置页中选择关闭该功能。

4. 日志与统计数据

本软件仅记录备份操作的必要日志（如文件数量、校验结果、耗时），存储在本地 SQLite 数据库中，用于在 Web 面板中展示历史记录。这些数据不会被自动上传或分享。

5. 开源审计

本软件以 MIT 协议在 GitHub 上开源，任何人均可审计源代码，确认数据处理逻辑。如果你发现任何隐私相关问题，欢迎通过 GitHub Issues 反馈。

6. 第三方服务

本软件在以下场景下会与第三方服务通信：百度网盘 API（需用户授权）、OpenAI 兼容 API（需用户配置）、GitHub（用于检查更新）。除此之外，本软件不会与其他第三方服务建立连接。

7. 联系方式

如有任何隐私相关问题，请在 GitHub 仓库提交 Issue，或联系作者：luguanlin20050927@iCloud.com
